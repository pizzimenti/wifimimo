# Support

## Where to ask for help

- Reproducible bugs: open a GitHub bug report.
- Feature ideas: open a GitHub feature request.
- Usage questions or setup help: open a GitHub support issue.
- Security reports: follow [SECURITY.md](SECURITY.md) and do not open a public issue.

## What to include

- The version from `VERSION`
- Linux distribution and kernel version
- KDE Plasma version, if the plasmoid is involved
- Wi-Fi adapter or chipset details when relevant
- Steps to reproduce, expected behavior, actual behavior, and logs or screenshots when available
